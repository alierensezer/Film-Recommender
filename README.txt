Film Recommender, TMDB API’den rastgele film önerisi çekip terminalde gösteren basit bir Python uygulamasıdır. Python, projenin arka planında veri çekme ve işleme için kullanılırken; web geliştirme kısmı HTML, CSS, JavaScript ve Node.js ile yapılmaktadır. Proje, DevOps süreçlerini de içine alarak otomatik güncellemeler için GitHub Actions ile entegre edilmiştir. Ayrıca Docker desteğiyle kolayca konteynerize edilebilir.

Özellikler
-Python ile TMDB API’den film verisi çekme ve işleme

-HTML, CSS, JavaScript ve Node.js ile frontend ve backend web geliştirme

-Haftalık otomatik güncelleme (GitHub Actions ile)

-Docker ile konteynerize çalışma

-DevOps süreçlerinin entegrasyonu

Kullanım
-Python scripti arka planda çalışarak veri oluşturur. Web uygulaması ise Node.js tabanlı backend ile bu veriyi sunar.

GitHub Actions
-Haftalık güncelleme için GitHub Actions workflow’u ekleyebilirsiniz.

Katkıda Bulunma
-Projeye katkı sağlamak için pull request gönderebilirsiniz.
